  • 良い感じの「余白」

  • 良い感じの「配色」

良い感じCSS

  • marginで外側の余白を確保
  • margin: 1em;
  • margin: 0 auto; /* よくやる中央寄せ */
  • paddingで内側の余白を確保
  • padding: 1em;
  • fontは良い感じに
  • font-family: "Meiryo UI";
  • font-size: 11pt;
  • background, colorで配色
  • background: #9ED5B7;
  • color: #666;
  • border-bottom: 1px solid #CCC;
  • 色が多くなりすぎないように注意！
  • box-shadowで†影†
  • box-shadow: 0 0 5px 0 rgba(0,0,0,0.5);
  • text-shadow: 0 0 3px rgba(0,0,0,0.3);
